# Callout Manager Local
本地修改版 Obsidian Callout 插件，使用独立插件 ID，避免被社区插件更新覆盖。

![Screenshot](docs/images/screenshot_manage_pane_darklight.png)

## Features

- **浏览可用 Callout。**
  在列表里查看可以使用的 Callout。

- **修改 Callout 颜色和图标。**
  按自己的需要调整 Callout 外观。

- **创建自定义 Callout。**
  没有合适的类型时可以自己创建。

- **自动检测主题和 CSS 片段中的 Callout。**
  插件会维护可用 Callout 列表。

- **快速插入默认 Callout。**
  默认命令会插入本地设置里的 Callout 类型。

- **独立本地 ID。**
  插件 ID 是 `callout-manager-local`，请安装到 `.obsidian/plugins/callout-manager-local`。
